## Kebutuhan
1. Install Python
2. Install Library yang diperlukan<br>
Masukan perintah berikut lewat prompt:

    pip install -r requirements.txt

## Cara run aplikasi
Masukan perintah berikut lewat prompt:

    python main.py wp

atau

    python main.py saw

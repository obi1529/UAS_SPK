## Kebutuhan
1. Install Python
2. Install Library yang diperlukan
    Masukan perintah berikut lewat prompt:
```pip install -r requirements.txt```
3. Install *[postman](https://www.postman.com/downloads/)*

## Cara run aplikasi
Masukan perintah berikut lewat prompt:
```python main.py```

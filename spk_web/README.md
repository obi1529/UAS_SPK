## Kebutuhan
1. Install Python
2. Install Library yang diperlukan<br>
Masukan perintah berikut lewat prompt:<br>

    pip install -r requirements.txt

3. Install *[Postman](https://www.postman.com/downloads/)*

## Cara run aplikasi
Masukan perintah berikut lewat prompt:

    python main.py
